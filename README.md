# Prodash

A personal productivity dashboard built with React, Vite, Supabase, and Resend.

## Features

- Calendar, Todos, Meetings, Notes, Reminders
- User authentication (sign up, sign in, password change)
- Email notifications via Resend
- Soft delete with Trash and restore
- Color-coded categories
- Dark theme

## Tech Stack

- **Frontend:** React + Vite
- **Database & Auth:** Supabase (PostgreSQL)
- **Email:** Resend
- **Hosting:** Vercel
